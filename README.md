# User Login Page

A simple and responsive Login Page built using HTML5 and CSS3. This project focuses on creating a clean and user-friendly login interface while practicing the fundamentals of web development and responsive design.

---

## Features

- Responsive login page layout
- Clean and modern user interface
- Email and password input fields
- Styled login button
- Organized HTML and CSS code
- Beginner-friendly project structure

---

## Technologies Used

- HTML5
- CSS3

---

## Project Structure

```text
user-Login_Page/
│── login.html
│── style.css
└── README.md
````

---

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/Shishir25/user-Login_Page.git
```

2. Open the project folder.

3. Open `login.html` in any modern web browser.
